# Price Calculator (JavaScript)

This project is a simple price calculator written in JavaScript.  
It calculates the total price of a single product and also the total cost of multiple fruits based on quantity and price.

## What this program does
- Prints product name, price, tax, and total cost
- Calculates total price including tax
- Calculates total cost for multiple fruits using quantity × price

## ▶️ How to Run
1. Save the file as `priceCalculator.js`
2. Open a terminal in the same folder
3. Run:

##bash
node priceCalculator.js
